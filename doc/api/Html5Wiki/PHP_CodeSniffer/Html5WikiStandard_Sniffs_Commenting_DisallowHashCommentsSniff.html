Class Html5WikiStandard_Sniffs_Commenting_DisallowHashCommentsSniff

This sniff prohibits the use of Perl style hash comments.

An example of a hash comment is:

  1.   # This is a hash comment, which is prohibited.
  2.   $hello = 'hello';
