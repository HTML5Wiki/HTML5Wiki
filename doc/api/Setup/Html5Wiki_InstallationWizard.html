Class Html5Wiki_InstallationWizard

An implementation of InstallationWizard.
It extends the default wizard with the step specifications and all necessary methods for the installation process.

install

true install( $stepData $stepData )

Does the following steps:

  • Create configuration
  • Setup the database with the schema and some default articles
  • Move the files from web/ one level up if necessary

If any of these steps could be executed, the user gets a report with detailed instructions what he has to do manually.
